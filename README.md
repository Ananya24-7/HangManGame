ğŸ® How to Play

A word has been randomly selected, and a hint is displayed at the top.
Click on the letters of the virtual keyboard to make a guess.
Correct Guess: The letter will be revealed in the word, and the keyboard button will turn green.
Wrong Guess: A part of the hangman will be drawn, and the keyboard button will turn red.
Win: Guess the entire word before the hangman drawing is complete.
Lose: If the hangman is fully drawn (6 wrong guesses), the game is over.
Click the "Play Again" button to start a new game with a new word.
